## Overview

This project demonstrates an end-to-end data analytics workflow using Python, SQL, PostgreSQL, and Power BI. It covers data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, interactive dashboard creation, and business reporting.

The objective is to transform raw data into meaningful insights that support data-driven decision-making.

## Project Workflow

### 1. Data Loading

1. Clone this repository.
2. Install the required Python libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn psycopg2
   ```
3. Open the Jupyter Notebook or Python script.
4. Load the dataset.
5. Run the data cleaning and EDA steps.
6. Import the cleaned data into PostgreSQL.
7. Execute the SQL queries.
8. Open the Power BI file to explore the dashboard.
9. Review the report and Gamma presentation.
